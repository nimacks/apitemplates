# Open Api Specification 

## Path 

Holds the relative paths to the individual endpoints and their operations.



#### Path Fields

- `path` * : A relative path to an individual endpoint (Path Item  object)
  


### Paths at the root Level
```javascript
paths:
  /posts:
    get:

  /posts:
    post:

  /posts:
    put:

  /posts:
    delete:
```
