# OpenApi Specification  

### Root Object Fields

This is the root document object of the OpenAPI document.

The following fields nodes are required

- openapi
- info
- paths

```YAML

openapi:

info:

servers:

paths:

components:

security:

tags:

externalDocs:

```

> #### Example

```YAML
openapi: "3.0.0"

info:

servers:

paths:

components:

security:

tags:

externalDocs:
```
